## project3 주제
- 유튜브의 댓글을 크롤링 하여 분류, 강서구청장의 선거 결과를 예측한다.

## project 사용언어 
- python(VScode, google colab)

## 수집 대상 [358,713개]
- 기간 : 2013.01. ~ 2023.11. (11년)
- 대상 : 출마 후보자 권수정, 김태우, 진교훈 관련 영상
- 분야 : 7개 분야 (동영상 제목 / 게시일  /영상 좋아요 수 / 댓글 / 작성자 / 댓글 작성일 / 댓글 좋아요 수)
